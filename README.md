# Ride Sharing System — OOP Demonstration (C++ & Smalltalk)

This repository contains two implementations of a simple Ride Sharing System designed to demonstrate OOP principles:
- Encapsulation
- Inheritance
- Polymorphism

## Contents
- `RideSharingapp.cpp` — C++ implementation (compile with g++)
- `rideshare.st` — Pharo/Squeak workspace code for Smalltalk
- `Developing a Class-Based Ride Sharing System.docx` — Short APA-7 style report (1–2 pages)


## How to run (C++)
```bash
g++ -std=c++17 RideSharing.cpp -o RideSharing
./RideSharing
